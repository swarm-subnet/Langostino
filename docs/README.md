# Docs folder

Quick reference index for all documentation in this folder.

## Overview

This folder contains all the documentation for setting up, configuring, and troubleshooting the Langostino drone system. Documents are organized by topic area.

## Documents Index

| Document | Description |
|----------|-------------|
| [**assembly/**](assembly/README.md) | Complete build guide and Bill of Materials for assembling the drone from scratch |
| [**SETUP_GUIDE.md**](SETUP_GUIDE.md) | Step-by-step ROS2 environment setup on Ubuntu 22.04/24.04 for Raspberry Pi |
| [**INAV_GUIDE.md**](INAV_GUIDE.md) | Flight controller configuration: PID tuning, MSP settings, navigation parameters |
| [**COMMANDS_GUIDE.md**](COMMANDS_GUIDE.md) | Quick reference for ROS2 launch commands, PM2 management, and node operations |
| [**CONFIG_PARAMS_GUIDE.md**](CONFIG_PARAMS_GUIDE.md) | Complete reference for all ROS2 node parameters in `swarm_params.yaml` |
| [**TROUBLESHOOTING_GUIDE.md**](TROUBLESHOOTING_GUIDE.md) | Common issues and solutions for hardware, software, and network problems |
